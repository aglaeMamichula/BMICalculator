# BMICalculator
## Introduction
This project makes it easy to see your BMI based on your height/ weight inputs.

## Installation
1. Ensure that you have Python installed on your PC.
2. Download the file "bmi_task.py".
3. Double-click the file to run.

## Usage
Simply enter your height in metres, and weight in kilograms to get your BMI!
